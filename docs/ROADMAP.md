# GuruDev® Roadmap

## v0.1 — MVP Semântico (April 2026) ✅

- [x] Lexer: tokenização básica (keywords, literais, comentários)
- [x] Parser: AST mínima (bind, tag, load, dispatch, display, evaluate)
- [x] Context Analyzer: propagação de clave e hermenêutica
- [x] Bytecode Generator: GuruAST → GuruByte (JSON)
- [x] GuruDVM: execução com 7 níveis hermenêuticos distintos
- [x] GuruMatrix 10×10 (Aristóteles × Domínios)
- [x] CLI (argparse + Click/Rich)
- [x] Testes: lexer, parser, DVM, integração
- [x] Documentação: README, SETUP, specs, whitepaper
