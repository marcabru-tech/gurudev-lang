# GuruDev Setup Guide

## Requirements

- Python 3.9+
- pip

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/marcabru-tech/gurudev-lang.git
cd gurudev-lang
```

### 2. Install in editable mode (with dev dependencies)

```bash
pip install -e ".[dev]"
```

This installs:
- `gurudev` CLI command
- All packages from `src/` (`compiler`, `runtime`, `gurumatrix`, `gurudev`, `cli`)
- Dev tools: `pytest`, `black`, `isort`, `mypy`, `ruff`

## CLI Usage

After installation, the `gurudev` command is available:

```bash
# Compile a .guru file to .gurub bytecode
gurudev build examples/mvp_demo.guru

# Run a .guru file at hermeneutic level 4
gurudev run examples/mvp_demo.guru --hermeneutica 4

# Legacy interface — run at all 7 levels (demo mode)
gurudev examples/mvp_demo.guru --demo

# Show generated bytecode
gurudev examples/mvp_demo.guru --gurubyte

# Semantic dry-run analysis
gurudev examples/mvp_demo.guru --dry-run
```

### Hermeneutic Levels

| Level | Mode        | Description                                  |
|-------|-------------|----------------------------------------------|
| 1     | LITERAL     | Raw string representation                    |
| 2     | ALEGÓRICO   | Typed structure with explicit metadata       |
| 3     | TÉCNICO     | Computational analysis (length, type, etc.)  |
| 4     | CONTEXTUAL  | Positioned in the GuruMatrix coordinate grid |
| 5     | COMPARATIVO | Semantic relations map                       |
| 6     | ANALÓGICO   | Cross-domain analogical connections          |
| 7     | ONTOLÓGICO  | Full ontological expansion in GuruMatrix     |

## Project Structure

```
gurudev-lang/
├── src/
│   ├── gurumatrix/      # Semantic coordinate space (10×10)
│   ├── compiler/        # Lexer → Parser → Bytecode generator
│   ├── runtime/         # GuruDVM execution engine + dispatch
│   ├── gurudev/         # CLI (click/rich), exceptions, logger
│   └── cli/             # gurudev_cli.py (argparse CLI)
├── tests/               # pytest test suite
├── examples/            # Sample .guru programs
├── docs/                # Specifications and documentation
└── scripts/             # Dev/release helper scripts
```

## Writing GuruDev Code

Basic syntax:

```guru
# Bind semantic context
bind clave = ciencia

# Load a resource
load formula_energia

# Dispatch with hermeneutic levels
dispatch hermeneutica formula_energia {
    case 1: display in context     # Literal mode
    case 4: display in context     # Contextual mode
    case 7: display in context     # Ontological mode
    default: display in context
}
```

## Troubleshooting

**ImportError: No module named 'compiler'**  
→ Run `pip install -e ".[dev]"` from the repo root.

**pytest: command not found**  
→ Install dev deps: `pip install -e ".[dev]"`

**GuruDev CLI not found after install**  
→ Check that your pip scripts directory is in your PATH.
